fe100-course
============

## server
- Run: $ node index
- Setup: $ npm install
- Visit: $ http://localhost:3000

## client/phase-1
- Run: $ gulp build
- Setup: $ npm install
- Tools: gulp, backbone, browserify

## client/phase-2
- Run: $ grunt serve
- Tests: $ grunt test
- Setup: $ npm install && bower install
- Tools: grunt, mocha, lodash, backbone

## client/phase-3
- Run: $ grunt serve
- Tests: $ grunt test
- Setup: $ npm install && bower install
- Tools: grunt, karma, jasmine, angular

## client/phase-4
- Run: $ npm run app
- Setup: $ npm install && bower install
- Tools: flux, react, webpack, babel-es6
